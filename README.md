# Semrush Batch Checker

Esta aplicação permite verificar o **Authority Score (AS)** e o **Tráfego Orgânico** de uma lista de domínios em massa utilizando a API do Semrush.

## Funcionalidades
- Verificação em massa de múltiplos domínios.
- Seleção de base de dados por país (BR, US, etc.).
- Barra de progresso em tempo real.
- Design premium com Glassmorphism.
- Exportação dos resultados em CSV.

## Como usar
1. Certifique-se de que você tem um servidor PHP rodando (ex: XAMPP, Laragon, ou `php -S localhost:8000`).
2. Acesse a pasta `semrush-checker` pelo seu navegador.
3. Cole a lista de domínios na caixa de texto.
4. Clique em "Iniciar Verificação".

## Configuração
A chave da API já está configurada no arquivo `api.php`.
